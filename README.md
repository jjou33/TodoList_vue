# TodoList_vue

vue 학습간 진행한 TodoList 개발 관련 저장소 입니다.

### 개요

---

`Vue` 의 `vuex` 학습을 위해 진행한 간단한 프로젝트입니다.

기술 스택

- `Vue`,`vuex` `Javascript`, `html`, `css`

### 실행 코드

---

1. 프로젝트 받기 / `git clone`
2. 구성 node module install / `npm install`
3. 로컬 실행 / `npm run dev`
